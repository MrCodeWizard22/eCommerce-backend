# 🛒 eCommerce Backend – Java Spring Boot

This is a backend API for an eCommerce platform built using **Spring Boot**. It provides functionalities for managing users, products, orders, categories, and authentication using JWT.

---

## 🚀 Features

- User Registration and Login (JWT Authentication)
- Role-based Access Control (Admin & User)
- Product Management (CRUD)
- Category Management
- Order Placement and Tracking
- Secure REST APIs
- Exception Handling
- Layered Architecture (Controller → Service → Repository)
- Spring Data JPA & MySQL integration

---

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring Security + JWT
- Spring Data JPA (Hibernate)
- MySQL
- Maven

---

## 🧪 API Testing

You can test the APIs using **Postman** .
